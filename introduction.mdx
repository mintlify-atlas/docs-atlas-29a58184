Gemma Gem is a Chrome extension that brings Google's Gemma 4 language model directly into your browser. Powered by WebGPU, it runs inference entirely on your device, giving you a capable AI assistant on every webpage you visit — with complete privacy.

Ask questions about what you're reading, have the AI interact with page elements, take screenshots, run JavaScript, and more — all without any of your data ever leaving your machine.

## Getting started

<Steps>
  <Step title="Build the extension">
    Clone the repo and run `pnpm install && pnpm build` to produce the extension bundle.
  </Step>
  <Step title="Load into Chrome">
    Open `chrome://extensions`, enable developer mode, and load the `.output/chrome-mv3-dev/` folder.
  </Step>
  <Step title="Download the model">
    Navigate to any page and click the gem icon. The model (~500MB for E2B) downloads and caches on first launch.
  </Step>
  <Step title="Start chatting">
    Once the gem icon turns solid green, the model is ready. Type a question or give a command.
  </Step>
</Steps>

<Note>
  Gemma Gem requires Chrome with WebGPU support. Most modern Chrome versions on desktop support this out of the box.
</Note>
